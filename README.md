# Sistema de Ficha Técnica

## Status do Projeto
Em desenvolvimento

## Tecnologias Utilizadas
- Java
- MySQL
- Git e GitHub

## Desenvolvedor
- Pedro Otavio de Jesus

## Objetivo do Software
O sistema tem como objetivo realizar o cadastro e gerenciamento de fichas técnicas de produtos, permitindo armazenar informações como composição do tecido, tipo de costura e processo de fabricação.

## Funcionalidades
- Cadastro de ficha técnica (INSERT)
- Edição de dados da ficha (UPDATE)
- Listagem de fichas cadastradas
- Integração com banco de dados
